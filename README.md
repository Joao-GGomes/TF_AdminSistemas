Exercício 1 – NGINX no Azure com Terraform e Ansible
Este repositório corresponde ao **Exercício 1** do trabalho final da unidade curricular de Administração de Sistemas.

O objetivo deste exercício é:
- Configurar automaticamente o NGINX utilizando Ansible
- Garantir que um ficheiro `index.html` válido é servido pelo NGINX
- Assegurar que não existem passwords ou credenciais guardadas nos ficheiros Terraform ou Ansible

Foram removidas todas as credenciais em texto plano dos ficheiros Terraform e Ansible. Foi garantido a funcionalidade do `index.html`.
